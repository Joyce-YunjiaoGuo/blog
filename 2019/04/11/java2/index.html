数据类型

–基本数据类型
–整形 byte/short/int/long
–浮点型 float/double
–字符型 char
–布尔型 boolean
–引用数据类型
–数组 array
–类 class
–接口 interface
–枚举 enum
–注解 @interface

基本数据类型

byte 字节型 8bit 00000000 第一位表示正负，后七位为数字范围 [-128, 127]
short 短整型 16bit 第一位表示正负，后15位为数字范围 [-2的15次方, 2的15次方-1]
int 整型 32bit 第一位表示正负，后31位为数字范围 [-2的31次方, 2的31次方-1]
long 长整形 64bit 第一位表示正负，后63位为数字范围 [-2的63次方, 2的63次方-1]

float 单精度 32bit 第一位表示正负，前9位表示整数部分，后22位表示小数部分
double 双精度 64bit 第一位表示正负，前19位表示整数部分，后44位表示小数部分

char 字符型 16bit Unicode编码 – 英文/数字/符号 1字节
– 中文 2字节

boolean 布尔型 1bit true/false

字符 vs 字符串

char    String
基本    引用
'a'     "abc","a","",null

常量&变量

常量 固定不变的值

–基本数据类型
–String类 eg: “abc”
–自定义 eg: final int UP = 1;

变量 内存空间中的值可变，但数据类型固定

声明变量时，必须指定 数据类型 和 变量名（小驼峰）eg: int a; String b;

数据类型转换

同种数据类型直接按可以直接赋值，但不同数据类型需要先转换再赋值。
