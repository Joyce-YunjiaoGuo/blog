树

一个树结构包含一系列存在父子关系的节点。每个节点都有一个父节点（除了root）以及零个或多个子节点。

节点：树中的每个元素。一个节点可以有祖先和后代。
根节点：位于树顶部的节点，没有父节点。

内部节点：至少有一个子节点的节点。
外部节点/叶节点：没有子元素的节点。

子树：由节点和它的后代构成。

节点的深度：取决于它的祖先节点的数量。
树的高度：取决于所有节点深度的最大值。

二叉树&二叉搜索树

二叉树Binary：二叉树中的节点最多只能由两个子节点，一个是左侧子节点，一个是右侧子节点。插入/查找/删除节点更高效。
二叉搜索树BinarySearchTree：只允许在左侧节点存储（比父节点）小的值，在右侧节点存储（比父节点）大（或者等于）的值。

BinarySearchTree, 向树中插入一个值

树的遍历

中序遍历：在二叉树中，中序遍历首先遍历左子树，然后访问根结点，最后遍历右子树。

前序遍历：在二叉树中，前序遍历首先遍历根节点，然后访问左子树，最后遍历右子树。

后序遍历：在二叉树中，前序遍历首先遍历左子树，然后访问右子树，最后遍历根节点。
