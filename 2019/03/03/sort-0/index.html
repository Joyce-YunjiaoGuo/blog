排序

冒泡排序 O(n²)

比较任何两个相邻的项，如果第一个比第二个大，则交换他们。元素项向上移动至正确位置的顺序，好像冒泡。

选择排序 O(n²)

找到最小值放置在第一位，找到第二小的值放置在第二位。。。

插入排序

每次排一个数组项。假定第一项已排好，第二项和其比较，应该插前面还是后面；接着第三项再比较…

归并排序 O（nlogn）

分治算法–将原始数组切分成较小的数组，直到每个小数组只有一个位置，接着将小数组归并成较大的数组，直到最后一个排序完的大数组。

快速排序 O(nlogn)

分治算法–将原始数组分为较小的数组，再一一处理完再合并
先选一个基准（首位或中间位），创建两个指针，从两头开始，直到左指针找到比基准大的，右指针找到比基准小的，然后交换他们。重复这步操作，直到左指针超过右指针。
=> 比基准小的都在左边，比基准大的都在右边。
对小数组重复以上两步操作，直至全部排序
